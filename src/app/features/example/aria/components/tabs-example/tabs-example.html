Tabs Example

A headless, accessible tabbed interface using @angular/aria. Use arrow keys to navigate between tabs.

Overview
Features
Accessibility

Overview Content

Angular Aria provides headless directives for building accessible interactive patterns. This implementation uses ngTabs, ngTabList, ngTab, and ngTabPanel.

Features Content

  • Headless logic (you control the styling)
  • WAI-ARIA pattern compliance
  • Keyboard navigation support
  • Focus management

Accessibility Content

By using these directives, you automatically get the correct ARIA attributes like role="tablist", role="tab", role="tabpanel", and state management like aria-selected and aria-controls.
